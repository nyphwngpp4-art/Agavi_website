# HVAC Toolkit

Back-office tooling for Agavi's Phoenix HVAC vertical.

Contents:
- `missed-call-audit.js` — generates the one-page Missed Call Audit report from a VoIP call log CSV
- `missed-call-audit-talk-track.md` — script for the 10-minute screenshare audit call
- `sample-call-log.csv` — example input for testing the script
- `monster-air-engagement.md` — engagement one-pager for the Monster Air pilot
